# TaskPilot Mobile QA Checklist

Test `/daily` at these widths:

- 390px (iPhone)
- 430px (large phone)
- 768px (tablet)
- desktop

Checks:

- No horizontal scroll on page body.
- Sticky compact header visible and readable.
- Current Mission appears above Copilot and queue.
- Copilot card visible without side-panel behavior.
- Long mission/checklist/proof text wraps cleanly.
- Bottom action bar appears when mission is active.
- Proof modal is mobile-friendly and save button stays reachable.
- Desktop column layout remains intact.
